SMTP and notifications

Auto deploy webhook success and failure results will be sent to this recipient when SMTP is configured.

Recommended flow

  1. Save panel listen address as `127.0.0.1:8080`.
  2. Save panel base URL as `http://panel.example.com`.
  3. Apply panel domain to create the Nginx reverse proxy config.
  4. Point DNS to this server.
  5. Enable panel TLS after DNS resolves correctly.
  6. After TLS works, update the panel base URL to `https://panel.example.com`.
  7. Restart the panel service if listen address changed.
  8. Save SMTP settings if you want auto deploy email results.
