Install Redis

This installs the Ubuntu `redis-server` package, enables the service at boot, and starts it immediately.

Credentials and port

The panel writes an ACL user, disables the default Redis account, stores the ACL file separately from the main config, and restarts Redis automatically after save.

Set `0` for unlimited memory, or choose an eviction policy if Redis should evict keys when it reaches the limit.

Service control

Start, stop, or restart the Redis service without leaving the panel.

Connection test

This runs `redis-cli PING` against `127.0.0.1` with the supplied ACL username, password, and port.

Redis logs

Open a separate page to inspect recent Redis service logs without crowding the control screen.

Open Redis logs

Current rules

  • Usernames must start with a letter and can contain letters, numbers, dashes, or underscores.
  • Passwords cannot be empty and cannot contain spaces.
  • Leaving the password field blank generates a new secret and shows it once after save.
  • Configuration changes are saved and Redis is restarted automatically.
  • Memory limit is written to `maxmemory` in bytes and eviction selection is written to `maxmemory-policy`.
