Rotate MySQL admin password

This updates the password of the MySQL admin account currently stored in the root-only defaults file and immediately rewrites that file.

Current rules

  • Names must start with a letter and contain only letters, numbers, and underscores.
  • The created user receives privileges only on the requested database.
  • If password is left blank, the panel generates a random secret and shows it once.
  • Admin password rotation updates only the saved MySQL admin credentials and does not change the panel application DSN.
