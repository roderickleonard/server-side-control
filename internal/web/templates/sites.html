Mode rules

  • Reverse proxy requires a domain and upstream target.
  • The panel always creates a root directory inside the selected Linux user's web directory.
  • Static site uses that generated root directly for file serving.
  • PHP site uses the generated root and requires a PHP-FPM version such as 8.2.
  • On failure, the manager rolls back the written file and leaves Nginx unchanged.
